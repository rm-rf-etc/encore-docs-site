Modules

Below are listed all of the framework modules that are available through the IoC module. In quotes are the names you should use to include them in any of your modules.

  • lodash-node: "_"
  • Runway: "router"
  • Easyioc: "ioc"
  • FileFetcher: "fetchfiles"
  • Encore RESTful Controller constructor: "RESTController"
  • Encore views store: "views"
  • Encore settings: "settings"
  • Node HTTP server object: "server"
