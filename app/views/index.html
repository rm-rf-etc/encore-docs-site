Encore

A modular server-side MVC for node.js

Hello World

// Your main.js
require('encore')(function(router, server){

    router('/', function(req, res, args, methods){
        res.end('hello world')
    })

    server.listen(8080) // Execute `node main.js`
}) // Browse to localhost:8080

Encore is a project to build a very clean and easy to read, easy to use server-side node.js framework, adhering to javascript best practices within the CommonJS module architecture. The primary motivating principles are:

  • DRY
  • SRP
  • Uncluttered
  • Convention over configuration.

Features Overview

  • Inversion of control dependency loading
  • Cleanest routing syntax ever
  • Route groups
  • Route wildcards
  • RESTful controllers
  • Filters on routes or controllers
  • View templating (any 3rd party templating library)
  • View rendering from memory but reloaded on file change.
  • Flexible recursive project loading
