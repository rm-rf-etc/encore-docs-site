Encore Project Status

Encore is in its infancy and needs contributors. The framework is composed mainly of independent modules, adhering to the unix philosophy.

Features still not addressed:

  • Automated response header writing and buffer length calculation.
  • Needs a clean and simple ORM?
    • Atop abstraction layer compatible w/ RDBMS's & future NoSQL engines(?)
    • Any good existing RDBMS ORMs for node? Bookshelf.js? Backbone-ORM?
    • Perhaps none should be pre-packaged?
  • Should have better error handling around view rendering.
  • Using view layouts could to be built-in? Should remain template engine agnostic.
  • Will eventually need a CLI tool or built-in REPL.
  • Currently lacks any features for auth, CSRF, etc. Include Passport or Connect?
  • Needs a module for email handling.
  • Front-end framework of same structure & simple abstracted communication layer?
    • Build process would run at launch, structure would be identical to backend.
    • Socket.io integration?
